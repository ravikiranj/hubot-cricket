# hubot-cricket

Fetch cricket scores

See [`src/cricket.coffee`](src/cricket.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-cricket --save`

Then add **hubot-cricket** to your `external-scripts.json`:

```json
[
  "hubot-cricket"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
